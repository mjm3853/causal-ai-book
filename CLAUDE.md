## Project Overview

This repository contains code and exercises for the "Causal AI" book by Robert Ness.

- Book: https://www.manning.com/books/causal-ai
- Resources: https://www.robertosazuwaness.com/workshops.html

## Development Setup

This is a Python project. Once dependencies are added:

```bash
# Install dependencies (using uv)
uv sync

# Run Jupyter notebooks
uv run jupyter lab

# Run tests
uv run pytest
```
